Function:

Other:

Warning: when you update verion to 0.8 beta 1 you must run script setup of version 0.8 later to remove

Dependence PyQt4:
- For Ubuntu: sudo apt-get install python-qt4-dev
- For Fedora: sudo yum install PyQt4-devel
